MODELO DE SITE PARA ACADEMIA COM QR CODE

COMO USAR:
1) Suba esta pasta para um repositório no GitHub.
2) Ative o GitHub Pages.
3) Coloque seus vídeos dentro da pasta /videos.
4) Mantenha os mesmos nomes dos arquivos ou altere no js/app.js.
5) Gere um QR Code para cada link.

EXEMPLOS DE LINKS:
- seuusuario.github.io/repositorio/?ex=supino
- seuusuario.github.io/repositorio/?ex=legpress
- seuusuario.github.io/repositorio/?ex=puxada

PARA ADICIONAR NOVO EXERCÍCIO:
- Abra js/app.js
- Copie um bloco dentro de EXERCISES
- Troque a chave, título, vídeo e textos

ARQUIVOS DE VÍDEO ESPERADOS:
- videos/supino.mp4
- videos/legpress.mp4
- videos/puxada.mp4
